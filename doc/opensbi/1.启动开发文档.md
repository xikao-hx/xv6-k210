# 1.任务目标
**在qemu上通过opensbi引导启动xv6**

**目录说明**：

* opensbi的目录对应opensbi-0.9
* xv6对应的目录是os
* opensbi的设备树是dts/xv6-k210.dtb，dts有内容是一些参考文件
* build.sh是编译脚本
* run.sh是启动qemu的脚本

**工具链**：
```shell
export ARCH=riscv  
export CROSS_COMPILE=riscv64-unknown-linux-gnu- 
export PATH=$PATH:/home/xikao/quard_star_tutorial/toolchain/gcc-riscv64-unknown-linux-gnu/bin/
```
